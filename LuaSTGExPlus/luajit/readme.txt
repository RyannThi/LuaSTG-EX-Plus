请到：http://luajit.org/download.html 下载luajit项目源代码并解压到本文件夹内
请确保你装有Visual Studio 2010（或2013、2015、2017、2019等），或者其他能编译luajit的工具

* 打开x86 Native Tools Command Prompt for VS 2010（或2013、2015、2017、2019等），如果你已安装Visual Studio系列产品，则该命令行工具将会附带于开始菜单内，请自行查找。
* 通过cd命令跳转到该目录内，如果跳转失败则可先切换盘符（如d:），再切换目录
* 通过cd命令进入到src文件夹内
* 输入msvcbuild启动批处理程序，接下来它将会自动编译生成所需的库文件和动态链接库文件

如果还有不明白的地方，可前往：http://luajit.org/install.html 查看官方文档，里面提供了详细的编译指南。
